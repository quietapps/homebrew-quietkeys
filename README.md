# Quiet Keys Homebrew tap

Homebrew tap for [Quiet Keys](https://github.com/quietapps/QuietKeys) — a native macOS menu bar app that plays realistic mechanical keyboard switch sounds on every keystroke and mouse click. Fully offline.

## Install

```sh
brew tap quietapps/quietkeys
brew install --cask quietkeys
```

## Update

```sh
brew update
brew upgrade --cask quietkeys
```

## Uninstall

```sh
brew uninstall --cask quietkeys
brew untap quietapps/quietkeys
```
